

# Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes sales data to identify trends, top-performing products, and loss-making areas.

## 🛠 Tools Used
- Power BI
- SQL
- Excel

## 📈 Key Insights
- East region has highest sales
- Technology category is most profitable
- High discounts lead to losses in some products

## 📉 Loss Analysis
Identified loss-making products and analyzed impact of discount on profitability.
